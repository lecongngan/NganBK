Câu lệnh thêm mới bản ghi :
        Cú pháp 1 :
        INSERT INTO table_name (column1, column2, column3, ...)
        VALUES (value1, value2, value3, ...);
        Cú pháp 2 :
        INSERT INTO table_name
        VALUES (value1, value2, value3, ...);
        Ví dụ thêm 1 bản ghi mới vào bảng customers ( cột id thì bỏ qua id sẽ tự sinh )
        ( cần cột nào thì thêm cột đó )
        INSERT INTO `customers` (`company`, `last_name`, `first_name`, `job_title`)
        VALUES ('Test', 'dat', 'do', 'dev')
